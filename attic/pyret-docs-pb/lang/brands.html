Brands

Usage:

include brands
import brands as ...

Brand<a>

Brands are a mostly internal language concept, useful for implementing custom datatypes.

brander :: () -> Brand<a>

Creates a new brand.

.brand :: Any -> a

Produce a copy of the value with this brand.

.test :: Any -> Boolean

Test if the value has this brand.
