10. GNU Info docs

You can call the macro

    .troff2page2info

within your document to signal to troff2page that it should also create an Info document.

If the basename of the input groff document is jobname, the Info file is created in jobname.info, which may be displayed on any Info viewer.

A good place to set .troff2page2info without modifying the input groff document is in .troff2pagerc or jobname.t2p, described on page 1.

If you’re sure you will always want an Info file for whatever document you give to troff2page, then you can set the environment variable TROFF2PAGE2INFO (the value doesn’t matter):

    export TROFF2PAGE2INFO=always
