7. Ebooks

To have your groff source be converted into an ebook-style HTML, set the number register t2pebook to non-0 using the command-line option

    -r t2pebook=1

This causes Troff2page to generate a single HTML page with no external CSS or image dependencies, and which can be sent to an ereader such as the Amazon Kindle.
